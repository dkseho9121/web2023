01. 데이터 이펙트 : 1부터 100까지 출력하기

for문을 이용해서 1~100까지 출력하는 예제입니다.

for(let i=1; i<=100; i++){
    
console.log(i)
}


//결과값 1,2,3,4,5,6 ~~~100

02. 데이터 이펙트 : 0부터 50까지 출력하기

for문을 이용해서 0~50까지 출력하는 예제입니다.

for(let i=0; i<=50; i++){

console.log(i)
}
    
    
//결과값 1,2,3,4,5,6 ~~~50

03.데이터 이펙트 : 1부터 100까지 짝수만 출력하기

for문을 이용해서 짝수를 출력하는 예제입니다.

for(let i=0; i<=100; i+=2){

console.log(i)
}

//증감 연산자를 이용하였습니다.
//i += 2는 i = i + 2와 같습니다. (2씩 증가합니다.)
//1 += 3 이면 (3씩 증가합니다.)
//1 += 4 이면 (4씩 증가합니다.)
    
//결과값 5,10,15,20,25 ~~~ 100

04.데이터 이펙트 : 1부터 100까지 5의 배수 출력하기

증감연산자를 사용하여 같은 결과값을 출력하는 예제입니다.
if문을 사용하여 같은결과값을 출력하는 예제입니다. 
//증감 연산자를 사용한 방법입니다.
for(let i=0; i<=100; i+=5){      //변수 1 선언 및 값을 1로 선언한뒤 i가 100보다
    console.log(i)
}
    


//if문을 사용한 방법입니다.
for(let i=1; i<=100; i++){       // 변수 i 선언 및 값을 1로 선언한뒤 i가 100보다 같거나 작을동안 명령문을 실행
    if( i % 5 == 0){                // if문을 사용하여 i를 5로 나누어준다.
        console.log(i)              // 출력한다.
    }
}

//결과값 5,10,15,20, ~~~100
