01.선언적 함수

선언적 함수(declarative function)는 자바스크립트에서 함수를 정의하는 방법 중 하나입니다.
{
    function func( ){
    document.write("실행되었습니다.");
    };
    func( );
}
결과 확인하기
실행되었습니다.!
함수를 선언할 때 function 키워드를 사용하여 정의하는 방법입니다. 함수 선언문 이라고도합니다.
선언적 함수는 함수를 정의하는 시점에서 함수 객체가 생성되며, 호이스팅(hoisting)이 가능합니다.
호이스팅이 가능하다는 것은 함수가 정의되기 이전에 함수를 호출할 수 있다는 것을 의미합니다.
따라서, 함수를 선언문으로 정의하면 코드의 가독성과 유지보수성을 높일 수 있습니다.

02. 익명 함수

변수는 데이터를 저장하는 "저장소" 이며 이 저장소에는 숫자, 문자, 함수, 객체 등을 저장할수 있습니다.
{
    const func = function( ){
        document.write("실행되었습니다.");
    };
    func( );
}
결과 확인하기
실행되었습니다.!
익명 함수는 함수 표현식(function expression)이나 콜백 함수(callback function) 등에서 주로 사용됩니다.
함수를 변수에 할당하거나, 객체의 속성 값으로 함수를 할당할 때 사용됩니다.

03. 매개변수 함수

매개변수 함수(parameter function)는 자바스크립트에서 함수의 인자로 전달되는 함수를 말합니다 
 {
    function func(str){
        document.write(str);
    };
    func("실행되었습니다.");
}
결과 확인하기
실행되었습니다.!
매개변수 함수를 이용하면 함수의 재사용성과 확장성을 높일 수 있습니다.
함수의 로직 중 일부를 매개변수 함수로 분리하여, 함수 호출 시에 동적으로 로직을 변경하거나 확장할 수 있기 때문입니다.

04. 리턴값 함수

리턴값 함수(return function)는 자바스크립트에서 함수가 실행된 결과를 반환하는 함수를 말합니다.
{
    function func(){
        const str = "실행되었습니다.";       
        return str;                        // return 키워드로 인해 결과 호출        
    };
    document.write(func());
}
결과 확인하기
실행되었습니다.!
함수 내부에서 연산, 조작, 처리한 결과를 반환하여, 이를 다른 변수나 함수에서 활용할 수 있습니다.

05. 화살표함수: 선언적함수

화살표 선언적 함수(arrow function declaration)는 자바스크립트에서 함수를 정의하는 방법 중 하나입니다.
{
    func = () => document.write("실행되었습니다.");
    // function func(){
    //     document.write("실행되었습니다.");
    // };
    func();
}
결과 확인하기
실행되었습니다.
일반적인 함수 선언문과 달리, 화살표 함수를 이용하여 간결하고 명확한 함수를 작성할 수 있습니다. 화살표 함수는 함수의 인자와 로직을 괄호로 묶어 화살표 표기법(=<)으로 구성됩니다.
화살표 함수는 일반적으로 함수 표현식(function expression)으로 사용되며,
변수에 할당되어 다른 함수의 인자로 전달되거나 콜백 함수(callback function)로 사용됩니다.

화살표 익명 함수(arrow anonymous function)는 자바스크립트에서 익명 함수를 작성할 때, 화살표 함수를 이용하는 것을 말합니다

    };
    // const func = function(){

    // };
    func();
}

실행되었습니다.
함수 이름이 없는 익명 함수를 작성할 때,
화살표 함수를 이용하여 간결하고 명확한 함수를 작성할 수 있습니다.
화살표 익명 함수는 함수의 인자와 로직을 괄호로 묶어 화살표 표기법(=<)으로 구성됩니다.
화살표 함수는 일반적으로 함수 표현식(function expression)으로 사용되며,

07. 화살표함수 : 매개변수

화살표 매개변수(arrow function parameters)는 자바스크립트에서 화살표 함수를 작성할 때, 함수 인자를 지정하는 방법을 말합니다.
{
    func = (str) => {
        document.write(str);
    };
    // function func(str){
    //     document.write(str);
    // };
    func("실행되었습니다.");
}
결과 확인하기
실행되었습니다.
일반적인 함수 선언문과 달리, 화살표 함수는 매개변수 선언부가 괄호로 묶이며, 매개변수가 하나일 경우 괄호를 생략할 수 있습니다.

08. 화살표 함수 : 리턴값 함수

화살표 리턴값 함수(arrow function with return value)는 자바스크립트에서 화살표 함수를 작성할 때, 함수의 반환 값을 지정하는 방법을 말합니다.
{
    func = () => {
        const str = "실행되었습니다.";
        return str;
    };
    // function func(){
    //     const str = "실행되었습니다.";
    //     return str;
    // };
    document.write(func());
}
결과 확인하기
실행되었습니다.
화살표 함수는 함수 본문 내부의 문장이 하나일 경우에는 중괄호와 return 키워드를 생략할 수 있으며,
이때는 해당 문장의 결과값이 함수의 반환 값으로 자동 지정됩니다.

09. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값

화살표 함수 익명 함수와 매개변수 리턴함수를 사용한 예제입니다
{
    const func = (str) => {
        return str;
    };
    document.write(func("실행되었습니다."));
}
결과 확인하기
실행되었습니다.
위 순대로 익명 화살표 함수 a를 선언하고, 이를 호출하여 "실행1"을 출력합니다.
상수(const) b에 익명 화살표 함수를 할당하고, 이를 호출하여 "실행2"를 출력합니다.
세 개의 문자열 매개변수를 받는 익명 화살표 함수 c를 선언하고, 이를 호출하여 "실행3실행4실행5"를 출력합니다.
매개변수와 인자는 쉼표로 구분되며, 여기서는 인자를 쉼표 없이 나열하였습니다.
세 개의 숫자 매개변수를 받는 익명 화살표 함수 d를 선언하고, 이를 호출하여 1, 2, 3을 더한 결과인 6을 출력합니다.
함수 내에서 return 키워드를 사용하여 값을 반환합니다.

결과 확인하기
1. 함수가 실행되었습니다.

3. 리액트가 실행되었습니다.

결과 확인하기

3. 리액트가 실행되었습니다.

        {
            num : 1,
            name : "함수",
            word : "실행"
        }, {
            num : 2,
            name : "자바스크립트",
            word : "실행"
        }, {
            num : 3,
            name : "리액트",
            word : "실행"

22. 함수 종류 : 즉시 실행 함수

즉시 실행 함수(IIFE, Immediately Invoked Function Expression)는 함수를 정의하고 즉시 실행하는 자바스크립트 패턴입니다.
이 패턴은 전역 스코프에 변수를 추가하는 것을 방지하고 모듈화된 코드를 작성하는 데 사용됩니다.

{
    function func( ){
        document.write("함수가 실행되었습니다.");
    };
    func( );

    (function ( ){
        document.write("500. 함수가 실행되었습니다.");
    })( );

    (( ) => {
        document.write("501. 함수가 실행되었습니다.");
    })( );
}
결과 확인하기
500. 함수가 실행되었습니다.
501. 함수가 실행되었습니다.
