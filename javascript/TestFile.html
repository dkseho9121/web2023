배열 메서드 : push()

push() 메서드는 배열 마지막 요소에 요소를 추가하고, pop() 메서드는 배열 마지막 요소를 삭제합니다.

const arrNum1 = [100, 200, 300, 400, 500];
const arrPush1 = arrNum.push(600);

console.log(arrNum1)    //[100, 200, 300, 400, 500, 600] //push 했으니까 넣어주는거고 pop해줬으니까는 빼주는거 고정도만알면됨

const arrNum2 = [100, 200, 300, 400, 500];
const arrPush2 = arrNum.pop();

console.log(arrNum2)
