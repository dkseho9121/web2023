배열 메서드 : join()

push() 메서드는 배열 요소을 문자열로 결합합니다. 반환은 문자열입니다.
const text1 = arrNum1.join('')      // 100200300400500
const text2 = arrNum1.join(' ')     // 100 200 300 400 500
const text3 = arrNum1.join('*')     // 100*200*300*400*500
const text4 = arrNum1.join('-')     // 100-200-300-400-500
const text5 = arrNum1.join('🎁')    // 100🎁200🎁300🎁400🎁500

console.log(text5)                  // 배열 요소를 문자열로 변환

배열 메서드 : push() / pop() : 반환(숫자)

push() 메서드는 배열 마지막 요소에 요소를 추가하고, pop() 메서드는 배열 마지막 요소를 삭제합니다. 
const arrNum = [100,200,300,400,500];
const arrpush = arrNum.push(600);       // 추가

console.log(arrNum);                    //[100,200,300,400,500,600]


const arrNum2 = [100,200,300,400,500];
const arrpush2 = arrNum2.pop();         //삭제

console.log(arrNum2);                   //[100,200,300,400]

배열 메서드 : unshift() / shift()

unshift() 메서드는 배열 처음 요소에 요소를 추가하고, shift() 메서드는 배열 처음 요소를 삭제합니다.

const arrNum = [100,200,300,400,500];
const arrpush = arrNum.unshift(600);         // 추가

console.log(arrNum);                         //[600,100,200,300,400,500]

const arrNum2 = [100,200,300,400,500];
const arrpush2 = arrNum2.shift();            //삭제

console.log(arrNum2);                        //[200,300,400,500]
