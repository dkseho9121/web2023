REACT

React는 페이스북이 개발한 오픈 소스 JavaScript 라이브러리로, 주로 웹 애플리케이션의 사용자 인터페이스(UI)를 구축하는 데 사용됩니다.

React의 주요 특징 중 하나는 가상 DOM(virtual DOM)입니다. 가상 DOM은 실제 DOM의 가벼운 복사본으로, 변경 사항이 발생할 때마다 전체 페이지를 다시 렌더링하지 않고 변경된 부분만 빠르게 업데이트할 수 있게 해줍니다. 이로 인해 React는 웹 애플리케이션의 성능을 크게 향상시킬 수 있습니다.

REACT 설치

node.js 설치
https://nodejs.org/

React를 설치하려면 Node.js와 npm(Node 패키지 매니저)이 필요합니다. Node.js가 설치되지 않은 경우, https://nodejs.org/에서 Node.js를 다운로드하고 설치하세요. Node.js를 설치하면 npm도 자동으로 설치됩니다.

npx create-react-app react1
